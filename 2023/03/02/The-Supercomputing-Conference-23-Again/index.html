# SC 的恩怨情仇

又一年的 SC 开始了，对于这个会呢，又爱又恨。作为实验室常青的会议，感觉大家对它的期待总是比其他的要更多，至今忘不掉当初刚刚去到无锡的时候，生物材料部上下一心，全员冲击 SC 的场景，那段时间每天晚上拼到 12 点，说累么，疲惫是有的，但是却全然没觉得多累。反倒是去年 SC22 的时候，整个部门充斥着松懈的氛围，除了自己一个人在瞎忙活，害怕断了断了实验室连续几年的 SC 成果，觉得负重前行，一方面觉得工作做的不错，一方面又觉得自己啥都不懂，反复拉扯的像一个皮筋。这和最开始去到无锡的时候形成完全不一样的对比的场景，说实话，当初那个忙碌的生物材料部，反而更让我觉得有着一种充实的感觉。SC22 收到的评审意见并不好，评价两极分化的严重，strong accept 和 strong reject 同时出现也是让我迷惑，自己翻文献尝试出来的算法，也让自己开心的不行，但是反而在评审时只字未提，真是奇怪的 reviewer 啊。

# SC23

新的一年的 SC23 又进到了 Abstract 的阶段，我认为那个工作是好的，我不觉得它不值得这篇会议，也不相信它发不出来，这么多的心血，足够让它成为一篇好文章，这是我一直以来的信心。段爷也联系我了，我准备把这篇文章继续投一下，这也是我想做的事情啊，即便自己因为诸多自己都想不明白的原因和事情放弃了一些事情，但是总归，时间会告诉我最好的选择和最棒的道路。过了一年，这篇文章都没怎么动，我觉得，要是这篇文章能羽翼丰满的面对新一轮 SC，那会发光的

# 文章

文章工作是对长程静电势的模拟算法优化，从算法改动，计算简化，主从掩盖，计算通信掩盖，模拟中断的框架，到流水排布，设计新的混洗策略，到尝试卷积计算算法，最后的效果也确实蛮不错，算是实现了自己当初立的 flag。希望你能发光吧，毕竟，你也不该在我手里蒙受灰尘的沾染。
